Ahora, debes modificar el pie de página de tu archivo principal (`index.html`) para que incluya los enlaces a estas nuevas páginas.

1.  **Abre** tu archivo `index.html`.
2.  **Busca** la sección del `<footer>` al final del archivo.
3.  **Reemplaza** el código actual del `<footer>` con este código actualizado:

#### Código para reemplazar en `index.html`


### **3. Actualización del CSS**

Finalmente, añade estas nuevas reglas a tu archivo `style.css` para asegurar que el pie de página se vea bien con los nuevos enlaces.

1.  **Abre** tu archivo `style.css`.
2.  **Añade** el siguiente código al final del archivo:

#### Código para añadir en `style.css`

```css
/* --- ESTILOS ADICIONALES FOOTER --- */
.main-footer .container {
    flex-wrap: wrap;
    gap: 20px;
}

.footer-left {
    text-align: center;
    flex-grow: 1;
}

.legal-links {
    margin-top: 10px;
    font-size: 0.9rem;
}

.legal-links a {
    color: var(--color-secondary-text);
    transition: color 0.3s ease;
}

.legal-links a:hover {
    color: var(--color-primary);
}

.social-links {
   text-align: center;
   flex-grow: 1;
}

@media (min-width: 768px) {
    .footer-left {
        text-align: left;
    }
    .social-links {
        text-align: right;
    }
}
```

Una vez hayas guardado todos los cambios, sube los **cuatro** archivos (`index.html`, `aviso-legal.html`, `politica-de-privacidad.html`, `politica-de-cookies.html`) y el `style.css` actualizado a tu repositorio de GitHub siguiendo los mismos pasos de `git add .`, `git commit` y `git push`. Tu sitio se actualizará automáticamente.
